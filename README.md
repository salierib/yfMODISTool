一款批量处理MODIS系列数据的工具，该工具基于ArcGIS的python工具箱实现：

该工具能够批量对MODIS原始数据文件（.hdf）进行提取子数据集、拼接、裁剪、投影栅格等操作，并根据选择的矢量文件裁剪得到对应区域的栅格文件（.tif）.
